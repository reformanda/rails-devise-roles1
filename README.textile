h1. USAASC Virtual Boards System

Built using the "Rails and Devise with Roles":http://railsapps.github.io/rails-devise-roles.  This is my first real rails project, so I appreciated all the help I received from others.

h4.  Features

*  runs with Jruby
*  target environment is Windows 2008 Server (with Tomcat 7)
*  uses Carrierwave for file uploads
*  pdf/merger for PDF merging
*  a ColdFusion service to convert DOCx file to PDF
